HTML

  • HTML gives content structure.
  • Head elements contain metadata that helps the browser to render the page correctly and are not visible to the user.
  • Body elements contain content shown to the user.
  • Elements that are listed inside other elements are called child elements.
  • Section elements are semantic elements that define a section in a document that contains thematic content.

    • CSS

  • CSS gives content style.
  • Inline CSS is used when a develope wants to style an element directly in the HTML file.
  • Internal CSS style sheets contain CSS rules for the webpage in the head section of the HTML file.
  • External CSS style sheets are the most common way to style a webpage and they allow developers to keep all their CSS rules in a separate file.
